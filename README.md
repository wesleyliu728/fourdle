# Fourdle

Fourdle is a wordle spinoff based off of the number four! Coded by Daniel Li and Wesley Liu


# To play:

Clone or download repository

Then cd into the repository

Run the command

```
npm start
```

and open a browser with

```
localhost:3000
```

and you are set :D
